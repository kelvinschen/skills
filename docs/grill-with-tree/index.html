§3

Command Signature

acpus workflow install <source> [options]

# source: local directory | .tar/.tar.gz/.tar.xz/.zip archive | HTTP(S) URL
# options:
  --project         Install into the project catalog (default)
  --global          Install into the global catalog
  --name <name>     Set the catalog entry name (inferred from source by default)
  --force           Overwrite an existing entry with the same name
  --rename <name>   Install under a different name when resolving a conflict
  --skip-check      Skip workflow check validation
  --type <type>     Set source type explicitly: directory|tar|zip|url
  --dry-run         Preview the operation without installing
§4

Process

This decision log was produced with Grill with Tree, a structured design-exploration method. Each session decomposes its target into decisions, terms, assumptions, risks, and scenarios; ranks them as P0 blockers, P1 high-risk choices, P2 normal choices, or P3 cleanup; and asks the decision maker batches of independent questions. Each option exposes its downstream effects. Chosen branches expand into new children, rejected branches are pruned, and the process continues until every P0/P1 node is resolved, defaulted, or explicitly deferred.
